# COVID-DatosGobAR
Análisis de datos sobre el impacto del Coronavirus en la República Argentina

## Origen de datos
Este dataset fue obtenido del sitio oficial de Datos del Gobierno de la República Argentina, a través del siguiente link: 
https://datos.gob.ar/dataset/salud-covid-19-determinaciones-registradas-republica-argentina
